YADNSS

A DragonNest skill simulator built on google-polymer

Try the demo

Features

  • Doesn't work in IE
  • Download a build as a picture
  • Build URL's for saving and distributing a skill build
  • Unobtrusive warning system
  • PvE and PvP skill information
  • Skill information up to level 100

Usage

  • Mouse over a skill to view its information in the info panel
  • Click/Right-Click a skill to increase/decrease its level
  • Shift-Click/Shift-Right-Click a skill to max/zero its level (Ctrl-Click works also)
  • Other features should be intuitive

Bugs

  • Doesn't work in IE
  • Warning system fails to detect class based sp requirements
